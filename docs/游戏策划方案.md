## 一、游戏概述

### 1.1 游戏名称
**Letter-Blast** - 字母快打

### 1.2 游戏类型
单词解谜游戏 (Word Search Puzzle) + 间隔记忆学习

### 1.3 核心玩法
在15×4的字母网格中，通过连接相邻格子（上下左右、对角线）拼写出单词。游戏开始时从词库中取出4个单词供玩家查找。游戏结合**莱特纳盒子（Leitner Box）间隔记忆法**，记录玩家的单词学习进度。

### 1.4 美术风格
采用**现代怀旧 2.5D 像素风**，结合复古课堂元素，营造青春校园怀旧氛围。详见：`docs/美术风格.md`

### 1.5 项目现状
项目已实现完整的核心功能：
- ✅ 15×4 网格生成与单词放置
- ✅ 鼠标拖拽输入与路径验证
- ✅ 单词验证系统（支持中英双语显示）
- ✅ 莱特纳盒子学习进度系统
- ✅ 得分与时间系统
- ✅ 提示系统（每局3次）
- ✅ 学习进度自动保存
- ✅ 开始界面与自定义词库导入
- ✅ 背景音乐支持

## 二、核心机制

### 2.1 网格系统

| 属性 | 值 |
|------|-----|
| 网格尺寸 | 15列 × 4行 |
| 格子尺寸 | 70×75 像素 |
| 连接方向 | 8方向（上下左右 + 对角线） |
| 最小单词长度 | 2个字母 |
| 每局单词数 | 4个 |

### 2.2 单词选择系统

**单词库**：当前包含19个水果类单词（带中文翻译）
```
APPLE(苹果), BANANA(香蕉), CHERRY(樱桃), DATE(枣), ELDERBERRY(接骨木果), 
FIG(无花果), GRAPES(葡萄), HONEYDEW(蜜瓜), KIWI(猕猴桃), LEMON(柠檬), 
MANGO(芒果), NECTARINE(油桃), PAPAYA(木瓜), QUINCE(木梨), RAISIN(葡萄干), 
STRAWBERRY(草莓), TANGERINE(橘子), ULUBAR(乌卢巴尔果), VANILLA(香草)
```

**单词选择规则**：
- 每局游戏从词库中选择4个单词
- 优先从莱特纳盒子中需要复习的单词选取
- 如果需要复习的单词不足4个，从词库补充新单词

### 2.3 时间与得分系统

**单局时长**：180秒（3分钟）

**得分规则**（加权计分）：
| 单词长度 | 基础分 | 难度系数 | 实际得分 |
|----------|--------|----------|----------|
| 2-3字母 | 10分 | ×1.0 | 10分 |
| 4-5字母 | 20分 | ×1.5 | 30分 |
| 6-7字母 | 40分 | ×2.0 | 80分 |
| 8+字母 | 80分 | ×3.0 | 240分 |

**时间奖励**：
- 每剩余1秒 = 0.5分
- 完美通关（剩余时间 > 50%）= 额外100分

### 2.4 游戏流程

```
开始界面 → 选择4个单词（优先复习） → 生成15×4网格 → 计时解谜 → 
找到所有单词 → 更新学习进度 → 显示得分 → 自动开始下一局
```

## 三、莱特纳盒子学习系统

### 3.1 系统概述

采用**莱特纳盒子（Leitner Box）间隔记忆法**管理单词学习进度。该系统通过5个盒子实现间隔复习，帮助玩家长期记忆单词。

### 3.2 盒子结构

| 盒子编号 | 复习间隔 | 说明 |
|----------|----------|------|
| Box 1 | 1天 | 新单词首次学习，第二天复习 |
| Box 2 | 2天 | 复习成功后进入，间隔2天复习 |
| Box 3 | 4天 | 复习成功后进入，间隔4天复习 |
| Box 4 | 7天 | 复习成功后进入，间隔7天复习 |
| Box 5 | 14天 | 复习成功后进入，间隔14天复习 |

### 3.3 学习规则

**新单词导入**：
- 新导入的单词统一放入 **Box 1**
- 记录首次学习日期

**复习晋升规则**：
- 复习成功（在游戏中正确找到单词）：单词晋升到下一个盒子
- 复习失败（未能在时间内找到）：单词降级到 Box 1

**单词选择优先级**：
1. Box 1 中超过1天未复习的单词
2. Box 2 中超过2天未复习的单词
3. Box 3 中超过4天未复习的单词
4. Box 4 中超过7天未复习的单词
5. Box 5 中超过14天未复习的单词
6. 如果需要复习的单词不足4个，从词库补充新单词

### 3.4 学习进度存储

**存储路径**：`user://learning_progress.json`

**存储结构**：
```json
{
  "version": "1.1",
  "last_updated": "2026-03-15T10:30:00Z",
  "total_words_learned": 15,
  "statistics": {
    "total_games_played": 42,
    "total_score": 12580,
    "words_mastered": 8
  },
  "boxes": {
    "box1": [{"word": "...", "first_learned": "...", "last_reviewed": "..."}],
    "box2": [],
    "box3": [],
    "box4": [],
    "box5": []
  }
}
```

#### `ui_manager.gd`

### 5.3 关键算法

#### 单词选择算法（莱特纳盒子）
```gdscript
func get_words_for_game() -> Array:
    var selected_words = []
    
    # 1. 按优先级遍历5个盒子
    for box_num in 1 to 5:
        var box_words = get_box_words(box_num)
        var interval = BOX_INTERVALS[box_num - 1]
        
        for word_data in box_words:
            if days_since(word_data.last_reviewed) >= interval:
                selected_words.append(word_data.word)
            if selected_words.size() >= 4:
                break
        if selected_words.size() >= 4:
            break
    
    # 2. 如果不足，补充新单词
    if selected_words.size() < 4:
        var new_words = get_new_words(4 - selected_words.size())
        selected_words.append_array(new_words)
    
    return selected_words
```

#### 单词放置算法（递归）
```
for 每个需要放置的单词:
    重复 MAX_PLACEMENT_ATTEMPTS 次:
        1. 随机选择起始坐标 (x, y)
        2. 验证首字母是否可放置
        3. 递归尝试构建路径
        4. 方向限制：8方向，禁止原路返回
        5. 支持字母共享（相同字母可重叠）
```

#### 路径验证算法
```
function is_valid_path(path):
    if path.length < 2: return false
    
    for i from 1 to path.length - 1:
        current = path[i]
        previous = path[i-1]
        
        if current in path[0:i]: return false  # 无自相交
        
        dx = abs(current.x - previous.x)
        dy = abs(current.y - previous.y)
        
        if dx > 1 or dy > 1: return false  # 必须相邻
        if dx == 0 and dy == 0: return false  # 不能原地
    
    return true
```
