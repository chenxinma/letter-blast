## 一、游戏概述

### 1.1 游戏名称
**Letter-Blast** - 字母快打

> **说明**: 本文档是游戏的完整策划方案，后续将根据此方案进行实现。当前项目已有基础框架，重点是完善 GameMechanics 和添加新功能。

### 1.2 游戏类型
单词解谜游戏 (Word Search Puzzle) + 间隔记忆学习

### 1.3 核心玩法
在4×18的字母网格中，通过连接相邻格子（上下左右）拼写出单词。游戏开始时从词库中取出4个单词供玩家查找。游戏结合**莱特纳盒子（Leitner Box）间隔记忆法**，记录玩家的单词学习进度。

### 1.4 美术风格
采用**现代怀旧 2.5D 像素风**，结合3D空间中的2D像素精灵与现代物理光照，营造青春校园怀旧氛围。详见：`docs/美术风格.md`

### 1.5 项目现状
项目已有基础框架（scripts/, scenes/, data/），实现了：
- ✅ 基础网格生成
- ✅ 单词放置算法
- ✅ 输入处理与路径验证
- ✅ 单词验证系统
- ✅ UI显示

待实现功能：
- ⏳ 莱特纳盒子学习进度系统

- ⏳ 得分系统优化
- ⏳ 声音系统

## 二、核心机制

### 2.1 网格系统

| 属性 | 值 |
|------|-----|
| 网格尺寸 | 4列 × 18行 |
| 格子尺寸 | 60×60 像素 |
| 连接方向 | 上、下、左、右（十字形，无斜向） |
| 最小单词长度 | 2个字母 |
| 最大单词长度 | 8个字母 |
| 每局单词数 | 4个 |

### 2.2 单词选择系统

**单词库**：当前包含21个水果类单词
```
APPLE, BANANA, CHERRY, DATE, ELDERBERRY, FIG, GRAPES, HONEYDEW, 
KIWI, LEMON, MANGO, NECTARINE, PAPAYA, QUINCE, RAISIN, 
STRAWBERRY, TANGERINE, ULUBAR, VANILLA
```

**单词选择规则**：
- 每局游戏固定从词库中选择4个单词
- 优先从莱特纳盒子中需要复习的单词选取
- 如果需要复习的单词不足4个，从词库补充新单词

### 2.3 时间与得分系统

**单局时长**：3-5分钟（平衡模式）

**得分规则**（加权计分）：
| 单词长度 | 基础分 | 难度系数 | 实际得分 |
|----------|--------|----------|----------|
| 2-3字母 | 10分 | ×1.0 | 10分 |
| 4-5字母 | 20分 | ×1.5 | 30分 |
| 6-7字母 | 40分 | ×2.0 | 80分 |
| 8+字母 | 80分 | ×3.0 | 240分 |

**时间奖励**：
- 每剩余1秒 = 0.5分（四舍五入）
- 完美通关（剩余时间 > 60秒）= 额外100分

### 2.4 游戏流程

```
游戏开始 → 选择4个单词（优先复习） → 生成4×18网格 → 计时解谜 → 
找到所有单词 → 更新学习进度 → 显示得分 → 开始下一局
```

## 三、莱特纳盒子学习系统

### 3.1 系统概述

采用**莱特纳盒子（Leitner Box）间隔记忆法**管理单词学习进度。该系统通过5个盒子实现间隔复习，帮助玩家长期记忆单词。

### 3.2 盒子结构

| 盒子编号 | 复习间隔 | 说明 |
|----------|----------|------|
| Box 1 | 1天 | 新单词首次学习，第二天复习 |
| Box 2 | 2天 | 复习成功后进入，间隔2天复习 |
| Box 3 | 4天 | 复习成功后进入，间隔4天复习 |
| Box 4 | 7天 | 复习成功后进入，间隔7天复习 |
| Box 5 | 14天 | 复习成功后进入，间隔14天复习 |

### 3.3 学习规则

**新单词导入**：
- 新导入的单词统一放入 **Box 1**
- 记录首次学习日期

**复习晋升规则**：
- 复习成功（在游戏中正确找到单词）：单词晋升到下一个盒子
- 复习失败（未能在时间内找到或错误）：单词降级到 Box 1

**单词选择优先级**：
1. Box 1 中超过1天未复习的单词
2. Box 2 中超过2天未复习的单词
3. Box 3 中超过4天未复习的单词
4. Box 4 中超过7天未复习的单词
5. Box 5 中超过14天未复习的单词
6. 如果需要复习的单词不足4个，从词库补充新单词

**文件路径**：`data/learning_progress.json`

### 3.5 单词学习记录字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| word | string | 单词内容 |
| first_learned | ISO8601 datetime | 首次学习时间 |
| last_reviewed | ISO8601 datetime | 最后一次复习时间 |
| review_count | integer | 总复习次数 |
| success_count | integer | 成功复习次数 |
| fail_count | integer | 失败复习次数 |

### 3.6 间隔计算逻辑

```gdscript
func get_words_for_review() -> Array:
    var current_time = Time.get_datetime_string_from_system()
    var words_to_review = []
    
    # Box 1: 1天间隔
    for word_data in boxes.box1:
        if days_since(word_data.last_reviewed) >= 1:
            words_to_review.append(word_data)
    
    # Box 2: 2天间隔
    for word_data in boxes.box2:
        if days_since(word_data.last_reviewed) >= 2:
            words_to_review.append(word_data)
    
    # Box 3: 4天间隔
    for word_data in boxes.box3:
        if days_since(word_data.last_reviewed) >= 4:
            words_to_review.append(word_data)
    
    # Box 4: 7天间隔
    for word_data in boxes.box4:
        if days_since(word_data.last_reviewed) >= 7:
            words_to_review.append(word_data)
    
    # Box 5: 14天间隔
    for word_data in boxes.box5:
        if days_since(word_data.last_reviewed) >= 14:
            words_to_review.append(word_data)
    
    return words_to_review
```

### 5.3 关键算法

#### 单词选择算法（莱特纳盒子）
```gdscript
func select_words_for_game(word_count: int = 4) -> Array:
    var selected_words = []
    
    # 1. 获取需要复习的单词
    var review_words = get_words_for_review()
    
    # 2. 按优先级排序（Box1 > Box2 > ... > Box5）
    review_words.sort_custom(func(a, b): return a.box < b.box)
    
    # 3. 选择复习单词
    for word_data in review_words:
        if selected_words.size() >= word_count:
            break
        selected_words.append(word_data.word)
    
    # 4. 如果不足，从词库补充新单词
    if selected_words.size() < word_count:
        var new_words = get_new_words_from_vocabulary(word_count - selected_words.size())
        selected_words.append_array(new_words)
    
    return selected_words
```

#### 单词放置算法
```
for 每个需要放置的单词:
    重复 MAX_PLACEMENT_ATTEMPTS 次:
        1. 随机选择起始坐标 (x, y)
        2. 随机选择方向 (0:下, 1:右, 2:上, 3:左)
        3. 检查是否可以放置（边界检查 + 冲突检查）
        4. 如果可以放置，则执行放置
    如果100次尝试后仍无法放置，调整网格或跳过
```

#### 路径验证算法
```
function is_valid_path(path):
    if path.length < 2: return false
    
    for i from 1 to path.length - 1:
        current = path[i]
        previous = path[i-1]
        
        if current in path[0:i]: return false  # 无自相交
        
        dx = abs(current.x - previous.x)
        dy = abs(current.y - previous.y)
        
        if dx > 1 or dy > 1: return false  # 必须相邻
        if dx == 0 and dy == 0: return false  # 不能原地
    
    return true
```

#### 得分计算算法
```
function calculate_score(word):
    length = word.length()
    
    if length <= 3:
        return 10 * 1.0
    elif length <= 5:
        return 20 * 1.5
    elif length <= 7:
        return 40 * 2.0
    else:
        return 80 * 3.0
```

#### 学习进度更新算法
```
function update_word_progress(word, success):
    word_data = find_word_in_boxes(word)
    
    if success:
        # 晋升到下一个盒子
        current_box = word_data.box
        if current_box < 5:
            move_word_to_box(word, current_box + 1)
        word_data.success_count += 1
    else:
        # 降级到 Box 1
        move_word_to_box(word, 1)
        word_data.fail_count += 1
    
    word_data.last_reviewed = current_datetime()
    word_data.review_count += 1
    
    save_progress_to_json()
```
