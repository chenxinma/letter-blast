# Letter Blast 游戏美术风格指引

## 一、核心风格定位
**风格名称**：复古像素课堂风
**核心气质**：怀旧、温馨、益智、略带复古手绘质感
**目标用户**：全年龄段，尤其适合喜欢复古像素风、单词益智类游戏的玩家

---

## 三、字体规范

### 1. 标题字体
- **字体选择**：复古像素字体（如 `Press Start 2P` 或定制像素字体）
- **应用场景**：游戏主标题 `Letter Blast`、关卡标题、结算页大标题
- **样式规范**：
  - 字号：48–64px
  - 颜色：深棕 `#5C4033`，描边 1px 浅金 `#FFD700`
  - 间距：字间距 4px，行间距 1.2 倍
  - 质感：模拟木质雕刻或黑板粉笔字效果

### 2. 正文字体
- **字体选择**：无衬线复古字体（如 `VT323`、`Courier Prime`）
- **应用场景**：分数、单词列表、提示信息、按钮文字
- **样式规范**：
  - 字号：18–24px
  - 字重：常规（400）或半粗（600）
  - 颜色：深棕 `#5C4033`，高亮时为金色 `#FFD700`
  - 行高：1.5 倍，确保可读性

### 3. 数字字体
- **字体选择**：等宽像素字体（如 `VT323`）
- **应用场景**：分数、倒计时、关卡编号
- **样式规范**：
  - 字号：24–36px
  - 颜色：深棕 `#5C4033`，描边 1px 浅金 `#FFD700`
  - 动画：分数更新时数字滚动并轻微放大

## 四、图标与UI组件规范

### 1. 图标风格
- **整体风格**：像素化+手绘质感，木质/纸张质感，圆角设计
- **图标尺寸**：
  - 功能图标：48×48px（如返回、设置、提示）
  - 道具图标：32×32px
  - 成就图标：64×64px
- **图标元素**：
  - 轮廓：1px 深棕 `#8B4513` 描边
  - 填充：米白 `#F5E6D3` 或浅金 `#FFE4B5`
  - 阴影：底部 1px 浅棕投影，营造轻微悬浮感

### 2. 核心UI组件
#### 分数面板
- 形状：木质边框矩形，圆角半径 8px
- 底色：米白 `#F5E6D3`，模拟纸张质感
- 边框：深棕 `#8B4513` 木质纹理
- 文字：像素字体，深棕 `#5C4033`，分数变化时闪烁金色

#### 单词列表面板
- 形状：木质边框矩形，圆角半径 8px
- 底色：浅绿 `#E9F5DB`，模拟旧纸张质感
- 边框：深棕 `#8B4513` 木质纹理
- 列表项：
  - 未找到：深棕文字 + 空复选框
  - 已找到：金色文字 + 勾选图标 + 浅黄高亮背景

#### 操作按钮
- 主按钮（如“开始游戏”“下一关”）：
  - 底色：浅金 `#FFE4B5`
  - 文字：深棕 `#5C4033` 半粗体
  - hover：底色加深 10%，轻微上浮
- 辅助按钮（如“提示”“重试”）：
  - 底色：米白 `#F5E6D3`
  - 文字：深棕 `#5C4033`
  - 描边：1px 深棕 `#8B4513`

## 五、动效与交互规范

### 1. 字母方块动效
- **选中状态**：
  - 缩放：1.05 倍
  - 发光：边缘出现 2px 金色辉光
  - 音效：清脆“咔哒”声
- **单词完成动画**：
  - 缩放：从 1.0 放大至 1.2 再缩回
  - 粒子：金色星星粒子向四周飘散
  - 时长：0.6s
- **消除/下落填充**：
  - 速度：匀速下落，距离越远速度越快
  - 缓冲：落地时轻微回弹，伴随“咚”声

### 2. 场景过渡动效
- **关卡切换**：
  - 淡入淡出：当前场景淡出（0.3s）→ 新场景淡入（0.3s）
  - 粒子：背景出现缓慢飘落的金色星星
- **结算页**：
  - 分数滚动：从 0 滚动至最终分数，时长 1s
  - 单词解锁：单词依次从列表底部弹出，伴随“叮”声

### 3. 反馈动效
- **正确操作**：
  - 文字：“Great!” 从顶部弹出，淡入淡出
  - 颜色：浅绿 `#4CAF50`
- **错误操作**：
  - 屏幕：轻微左右抖动
  - 颜色：浅红 `#FFCDD2` 闪烁 0.2s
- **成就解锁**：
  - 弹窗：从顶部滑入木质成就卡片
  - 粒子：金色星星粒子向四周扩散

## 六、场景与布局规范

### 1. 主游戏场景
- **视角**：等距俯视视角，模拟桌面视角
- **核心布局**：
  - 中心聚焦：字母棋盘占据画面 60% 区域
  - 边距：上下左右各留 10% 空白，避免拥挤
- **背景元素**：
  - 教室墙面：浅米黄 `#E9EDC9`，搭配复古肖像画、十字架装饰
  - 桌面：暖木色 `#D4A373`，带有光影纹理和盆栽装饰
  - 黑板：深灰黑 `#333333`，作为场景背景元素

### 2. 信息分区
| 区域 | 内容 | 布局优先级 |
| :--- | :--- | :--- |
| 左上角 | 分数面板 | 次要 |
| 左下角 | 单词发现面板 | 次要 |
| 右侧 | 本关单词列表 | 次要 |
| 中心 | 字母棋盘 | 核心 |

---

## 七、质感与细节规范

### 1. 材质表现
- **木质元素**：木纹纹理，哑光质感，边缘轻微磨损
- **纸张元素**：做旧纹理，轻微泛黄，边缘卷边效果
- **字母方块**：哑光纸质质感，表面无反光，边缘轻微圆角

### 2. 细节点缀
- 光影：桌面带有自然光影，模拟午后教室阳光
- 装饰：盆栽、肖像画、十字架等复古元素，增强场景沉浸感
- 交互反馈：所有可交互元素 hover 时均有颜色或阴影变化，提升可感知性

---

## 八、设计目标与一致性原则
1.  **色彩一致性**：所有元素配色严格遵循复古课堂色系，避免突兀色彩
2.  **风格统一性**：像素风与手绘质感融合，保持怀旧课堂氛围
3.  **可读性优先**：字体大小、颜色对比度需满足 WCAG 2.1 AA 级标准
4.  **反馈即时性**：所有玩家操作均需有明确的视觉/听觉反馈，降低学习成本
